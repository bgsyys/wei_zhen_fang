在线点餐平台项目

#### 项目介绍
在线点餐平台后端采用spring boot框架，数据库采用Mysql，利用Redis做缓存，Maven版本控制，前端采用Vue框架。
本项目包括后台管理端和用户端两部分，实现了后台管理系统中用户登录以及对员工，菜品，套餐管理的功
能，实现了移动端小程序在线浏览菜品，套餐，添加购物车及下单功能。

#### 使用说明

1.  java17版本
2.  Maven3.8.9版本

2.功能介绍
用户端管理登录采用JWT令牌登录方式一次登录后访问其他页面时无需再次输入用户名和密码，与Cookie与Session相比更加安全和方便，Cookie是存储在浏览器的数据每次访问服务端需要带着这条数据来判断是非需要登录，可以直接在浏览器中查询到对应的数据；Session是保存在服务器端的，当输入用户名和密码之后会转化成对应的SessionID和过期时间以Cookie形式保存并发送给用户端，当用户端访问服务时只需带着Cookie即可，但是当有大量用户同时访问时，可能会导致服务器过载从而Session失效；而JWT令牌登录，会将用户输入的用户名和密码进行JWT签名密文签名保存在Cookie中发送给用户，此时Cookie是保存在用户端的，每次访问只需带着对应的Token即可访问各个页面。
![输入图片说明](img/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-01-25%20224814.png)
